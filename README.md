# 大学生心理健康课程 - 心理测试平台

一个专为大学生心理健康课程设计的心理测试平台，提供多种经典心理测评工具，帮助学生更好地了解自己。

## 🎯 项目简介

本项目包含以下心理测试工具：

### 1. 霍兰德职业兴趣测试 (Holland Code Career Test)
- **理论依据**: 约翰·霍兰德（John Holland）的职业选择理论
- **测试维度**: 现实型(R)、研究型(I)、艺术型(A)、社会型(S)、企业型(E)、常规型(C)
- **适用场景**: 职业规划、专业选择、就业指导

## 📊 霍兰德职业兴趣测试说明

### 测试内容
- 共60道题目，涵盖6种职业兴趣类型
- 每题5个选项，从"非常不喜欢"到"非常喜欢"
- 预计完成时间：10-15分钟

### 六种类型详解

| 类型 | 英文 | 特点 | 适合职业 |
|------|------|------|----------|
| 现实型 | R (Realistic) | 动手能力强，喜欢具体任务 | 工程师、技术员、建筑师 |
| 研究型 | I (Investigative) | 善于分析，喜欢探索理解 | 科学家、研究员、医生 |
| 艺术型 | A (Artistic) | 富有创造力，追求自我表达 | 设计师、作家、音乐家 |
| 社会型 | S (Social) | 乐于助人，善于沟通 | 教师、咨询师、社工 |
| 企业型 | E (Enterprising) | 领导力强，善于说服 | 经理、销售、律师 |
| 常规型 | C (Conventional) | 细致认真，喜欢条理性 | 会计、行政、数据分析 |

### 结果解读
测试完成后，系统会生成：
1. **三字母代码**: 代表你最突出的三种兴趣类型（如：RIA、SEC等）
2. **雷达图**: 直观展示六种类型得分
3. **职业建议**: 根据代码推荐适合的职业方向
4. **详细分析**: 各维度得分解读
